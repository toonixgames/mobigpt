# MobiGPT v1.0.0

## Initial Release

MobiGPT is a mobile AI chat application that brings powerful local AI models directly to your device. Chat privately without an internet connection using state-of-the-art language models.

### Key Features

- **Local AI Models**: Run AI models completely offline on your device
- **Multi-Chat Sessions**: Manage up to 10 separate chat conversations
- **Image Support**: The model support image input for multimodal interactions
- **Function Calling**: Advanced models support function calling capabilities
- **Bilingual Interface**: Full support for Hebrew and English with RTL layout
- **Model Management**: Easy download and management of AI models

### Supported Models

- Gemma 3 Nano (2B, 4B) - Multimodal with function calls

### Platform Support

- Android

---

**Note**: Models must be downloaded separately. The app will guide you through the download process for your selected model.
